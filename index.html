01 / AI-квалификация лидов
Бот берёт на себя 80% входящих диалогов — менеджер получает уже квалифицированный лид
80%
Входящий поток обрабатывался вручную — даже рутинные вопросы про цену или расписание занимали время менеджеров. Внедрил AI-бота через Suvvy.ai с интеграцией в amoCRM и Wazzup. Бот самостоятельно ведёт первичный диалог, собирает нужные данные, заполняет карточку сделки и передаёт менеджеру уже квалифицированный лид. Первый ответ — в течение 15 секунд, 24/7. Менеджеры сосредоточились на сделках, а не на первичных переписках.

02 / Автоматический анализ звонков
Каждый звонок расшифрован и оценён по скрипту — за меньше $5 в месяц
<$5/мес
Контроль качества звонков традиционно требует либо отдельного специалиста, либо времени руководителя. Собрал систему на n8n + AssemblyAI + DeepSeek: запись звонка из CRM автоматически расшифровывается, затем AI сравнивает текст с фиксированным скриптом продаж и выставляет оценку по каждому пункту. Итоговый разбор с баллами и резюме приходит в Telegram через несколько минут после завершения разговора. Стоимость всей инфраструктуры — меньше $5 в месяц. Посмотреть дашборд →

03 / Аналитика вместо KPI
Менеджеры на чистой комиссии стали продавать больше — без принудительных показателей
↑ Конверсия
Менеджеры работали только на комиссии, без оклада — вводить KPI в такой модели сложно и рискованно. Вместо этого собрал в Google Sheets детальный аналитический файл: воронка по лидам, конверсия на каждом этапе, эффективность источников — отдельно для каждого сотрудника. Когда человек сам видит, где именно теряются деньги — он начинает это исправлять. Конверсия выросла без административного давления.

04 / Сквозная аналитика
Еженедельный отчёт для руководства: с 1–2 часов ручной сборки до 15 минут автоматически
−95% времени
Еженедельный отчёт для руководства собирался вручную из amoCRM, Яндекс.Метрики и Яндекс.Директа — уходило 1–2 часа и периодически возникали расхождения в данных. Написал скрипт на Google Apps Script, который автоматически запрашивает API всех трёх сервисов, сводит данные в единую таблицу и строит дашборд. Отчёт формируется за 15 минут без участия человека — данные всегда актуальны и согласованы.

05 / Аудит SaaS-расходов
Провёл аудит сервисов и срезал 30% лишних трат без потери функционала
−30% / год
Провёл аудит всех используемых SaaS-сервисов: реальная нагрузка, пересечения по функционалу, дублирующие инструменты. Часть платных интеграций перенёс на n8n — self-hosted решение, которое уже было развёрнуто на своём сервере. Ряд задач закрыл через Google Apps Script. По итогам аудита ежегодные расходы на инструменты сократились на 30% без потери функциональности.

06 / Мультиканальная воронка
WhatsApp, Telegram, звонки, email и SMS — единая воронка в amoCRM
Omnichannel
Входящие обращения приходили через разные каналы, каждый из которых существовал отдельно — менеджеры вручную переключались между интерфейсами. Собрал единую коммуникационную инфраструктуру: Wazzup и whatCRM для мессенджеров, SIPUNI и Mango для телефонии, DashaMail для email, smsc.ru для SMS. Все входящие автоматически попадают в карточку сделки в amoCRM. Настроены триггерные цепочки — нужный шаблон отправляется клиенту в нужный момент без участия менеджера.
amoCRMWazzup whatCRMSIPUNI / Mango DashaMailsmsc.ru
07 / Транскрибация и резюме встреч
Запись встречи — текст, тип встречи и краткое резюме — автоматически
Auto
Записи встреч накапливались в папке без использования — прослушивать часовые записи не было ни времени, ни ресурса. Настроил воркфлоу на n8n: файл попадает в Google Drive, расшифровывается через AssemblyAI, OpenAI определяет тип встречи и формирует краткое резюме, результат сохраняется в Google Sheets. Руководитель получает суть встречи за 30 секунд. Система полностью автономна.
n8nAssemblyAI OpenAI APIGoogle Drive Google Sheets
08 / Умное распределение лидов
Лид автоматически находит нужного менеджера — повторные назначения и дубли исключены
1 500 лидов
При высоком потоке лидов возникали ситуации: один лид назначался повторно или попадал к менеджеру, который уже работал с этим контактом. Построил систему распределения на n8n: по команде через Telegram воркфлоу обходит все незакрытые лиды (до 1 500), проверяет историю событий в amoCRM по каждому и назначает только на менеджера, который ранее не касался этого контакта. Дополнительно — автоматически закрывает дубликаты сделок по одному контакту, сохраняя самую раннюю.
n8namoCRM API Telegram BotPostgreSQL Google Sheets
09 / Автоматические напоминания о встречах
За час до встречи менеджер получает уведомление в Telegram — пропущенных встреч больше нет
0 пропусков
Задачи в CRM создавались, но менеджеры не всегда успевали их проверить — встречи иногда пропускались. Реализовал воркфлоу на n8n, который по расписанию опрашивает amoCRM, находит задачи с типом «встреча» и отправляет персональное уведомление в Telegram ровно за час до начала. Отдельная цепочка — уведомление при создании новой сделки. Настроено на несколько получателей, легко расширяется.
n8namoCRM API Telegram BotGoogle Sheets
10 / Валидация 200 000 email-адресов
Прошёлся по 86 рассылочным спискам и автоматически вычистил невалидные адреса
200k emails
База рассылки накопила 200 000 адресов в 86 списках — часть из них были невалидными, одноразовыми или неактивными, что снижало доставляемость. Написал скрипт на Google Apps Script с интеграцией DashaMail API: загружает все контакты, проверяет каждый адрес по 8 критериям (синтаксис, DNS/MX-записи, одноразовые домены, чёрные списки и другие), формирует отчёт с разбивкой по спискам. Удаление выполняется через чекбоксы — без риска случайно затронуть нужные контакты.
